
# Sysmon

## Overview

Sysmon is a Windows system monitoring tool that provides detailed event logs for process creation, network connections, and system activity.

---

# Use Cases

- Process monitoring
- PowerShell detection
- Network connection analysis
- Persistence detection
- Threat hunting

---

# Skills Practiced

- Event ID analysis
- Parent-child process investigation
- PowerShell monitoring
- Registry monitoring
- Process execution tracking

# Important Event IDs

| Event ID | Description |
|---|---|
| 1 | Process Creation |
| 3 | Network Connection |
| 7 | Image Loaded |
| 11 | File Create |
| 13 | Registry Modification |
