
# VirusTotal

## Overview

VirusTotal is a threat intelligence platform used to analyze suspicious files, URLs, domains, and IP addresses.

---

# Use Cases

- Hash reputation analysis
- URL analysis
- Domain reputation checks
- Malware identification
- IOC validation

# Skills Practiced

- Threat intelligence analysis

1. Extract IOC

3. Review detection results
4. Analyze relationships
5. Identify malicious indicators
