
# Wireshark

## Overview

Wireshark is a network protocol analyzer used to capture and inspect network traffic in real time.

---

# Use Cases

- Packet analysis
- DNS investigation
- HTTP/HTTPS traffic inspection
- Malware traffic analysis
- Suspicious network activity detection

---

# Skills Practiced

- Filtering packets
- Identifying suspicious connections
- TCP stream analysis
- Detecting beaconing traffic
- Analyzing DNS queries

---

# Common Filters

```bash
ip.addr == 192.168.1.1
tcp.port == 443
dns
http
```

---

# Investigations Performed

- Malware Traffic Analysis
- DNS Tunneling Detection
- Suspicious HTTP Requests
- C2 Communication Detection

---

# Key Learnings

- Understanding network protocols
- Identifying malicious communication patterns
- Detecting abnormal traffic behavior
